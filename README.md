# Jithin & Safna — Wedding Invitation Website

A beautiful, royal Mughal-themed digital wedding invitation for the reception of **Jithin Abdulsalam** and **Safna Latheef**, taking place on **Sunday, 16 August 2026** at **Everest Convention Centre, Karikkad, Thrissur, Kerala**.

## Features

- **Royal Mughal Aesthetic** — Deep burgundy & gold palette inspired by the Mughal era, with Islamic geometric patterns, Mughal arch motifs, and Arabic typography
- **Live Countdown Timer** — Real-time countdown to the wedding day (11:00 AM IST, 16 August 2026)
- **Add to Calendar** — One-click Google Calendar integration and .ics download for Apple Calendar / Outlook
- **Map & Directions** — Embedded Google Maps showing the venue with a direct "Get Directions" button
- **Family Details** — Both families (groom's from Thrissur, bride's from Palakkad) presented elegantly
- **Scroll Reveal Animations** — Smooth entrance animations for each section
- **Fully Responsive** — Works beautifully on mobile, tablet, and desktop

## Tech Stack

- **TanStack Start** (React + Vite) — Full-stack React meta-framework
- **TanStack Router** — File-based routing with type safety
- **Tailwind CSS v4** — Utility-first styling (used minimally; most styling is custom CSS)
- **Google Fonts** — Cinzel Decorative, Cormorant Garamond, Amiri (Arabic), Great Vibes
- **Netlify** — Hosting and deployment

## Running Locally

```bash
npm install
npm run dev
```

The dev server starts on `http://localhost:3000`.

For full Netlify feature emulation:

```bash
netlify dev --port 8889
```
